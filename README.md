# Айнүр қыз ұзату

Статичный одностраничник для GitHub Pages.

## Музыка

Положите свой файл рядом с `index.html` и назовите его `music.mp3`.
Если нужен другой путь, измените `src="music.mp3"` в `index.html`.

## Google Таблица

1. Создайте Google Таблицу с колонками: `имя`, `выбор`, `количество человек`.
2. Откройте `Расширения -> Apps Script`.
3. Вставьте код из `google-apps-script.gs`.
4. Нажмите `Deploy -> New deployment -> Web app`.
5. В `Who has access` выберите `Anyone`.
6. Скопируйте URL веб-приложения.
7. В `script.js` вставьте URL в `googleScriptUrl`.

После этого форма будет отправлять ответы в таблицу.
